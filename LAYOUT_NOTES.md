## 关键特性

### 响应式设计
- **桌面版** (> 1024px)：完整布局，所有元素可见
- **平板版** (768px - 1024px)：按钮尺寸减小，图标隐藏
- **手机版** (< 768px)：按钮更小，文字隐藏，仅显示图标

### 自适应滚动
- 一级分类行和二级分类行都支持水平滚动
- 当内容超过容器宽度时自动显示滚动条
- 滚动条样式与主题一致

### 颜色主题
- 背景色：深紫黑 (#0f0a1a)
- 表面色：#1a1428
- 卡片色：#2a1f3d
- 边框色：#3d2f52
- 主紫色：#a78bfa
- 浅紫色：#c4b5fd
- YES 按钮：#10b981 (绿色)
- NO 按钮：#ef4444 (红色)

## 交互功能

### 一级分类切换
1. 用户点击一级分类按钮
2. 按钮变为激活状态（紫色背景）
3. 前端调用 API 获取该分类的二级分类
4. 二级分类行更新显示新的分类

### 二级分类选择
1. 用户点击二级分类按钮
2. 按钮变为激活状态
3. 市场列表根据选中的分类过滤（待实现）

### 排序和过滤
1. 用户选择排序方式（交易量、概率、最新）
2. 用户选择状态过滤（全部、交易中、已结束）
3. 市场列表实时更新
