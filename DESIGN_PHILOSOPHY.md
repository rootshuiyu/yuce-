# Super Truth 预测市场平台 - 设计理念

## 设计方向：信息聚合型极简主义

### 核心设计原则

1. **数据优先** - 市场数据和价格信息是核心，UI 应该清晰展示隐含概率
2. **极简透明** - 避免过度装饰，让用户快速理解市场状态
3. **信息层级** - 通过颜色、大小和位置清晰传达重要信息
4. **实时感** - 使用动画和过渡效果传达市场的动态性

### 色彩系统

- **主色** - 深蓝色 (#1e40af) - 代表信任和专业性
- **成功色** - 绿色 (#16a34a) - 代表 YES 选项
- **警告色** - 红色 (#dc2626) - 代表 NO 选项
- **中立色** - 灰色系 - 用于背景和次要信息
- **强调色** - 琥珀色 (#f59e0b) - 用于重要操作

### 排版系统

- **标题字体** - Geist Sans (Bold/700) - 现代、清晰
- **正文字体** - Inter (Regular/400) - 易读、专业
- **数据字体** - Fira Code (Mono) - 用于价格和地址显示

### 布局范式

- **网格系统** - 12 列响应式网格
- **间距** - 8px 基础单位
- **卡片设计** - 柔和阴影、圆角 8px
- **非对称布局** - 侧边栏 + 主内容区

### 交互设计

- **悬停效果** - 轻微提升和背景变化
- **加载动画** - 脉冲效果表示实时更新
- **过渡时间** - 150-300ms 的流畅过渡
- **反馈** - Toast 通知用户操作结果

## 页面结构

### 1. 首页 (Markets List)
- 顶部导航栏
- 搜索和筛选面板
- 市场卡片网格
- 市场卡片显示：问题、隐含概率、交易量、关闭时间

### 2. 市场详情页 (Market Detail)
- 市场信息头部
- 订单簿显示 (YES/NO)
- 交易历史
- 用户持仓

### 3. 用户界面
- 钱包连接
- 用户资产
- 交易历史
- 排行榜

## 实现细节

- 使用 shadcn/ui 组件库
- Tailwind CSS 进行样式
- Framer Motion 用于动画
- Recharts 用于图表展示
